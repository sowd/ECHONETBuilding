# ECHONETBuilding
 A building emulator with lots of ECHONET Lite devices

BEMSでもECHONET Liteが役に立つのではという考えから作っている、ビルを意識した機器エミュレータです。OpenECHO使用。
画面サイズは800x480です。

## Updates

2017/9/18 MoekadenRoomからいろいろ流用して初期バージョン。一応機器オブジェクトは残している。
